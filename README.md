## Аналіз ринку нерухомості на платформі Airbnb у Нью-Йорку за 2019 рік

Об’єкт дослідження: Відкритий набір даних [AB_NYC_2019](https://www.kaggle.com/datasets/dgomonov/new-york-city-airbnb-open-data), що охоплює понад 48 000 записів про житло, доступне для оренди у п’яти ключових районах Нью-Йорка.

Мета: Виконати комплексний аналіз даних, щоб виявити основні закономірності у ціноутворенні, популярності районів та особливостях поведінки орендодавців.

Основні етапи реалізації:

- Очищення та підготовка даних: Пошук і усунення аномалій, таких як нульова доступність чи цінові викиди та створення очищеного набору даних.

- Географічний аналіз: Порівняння п’яти районів міста та детальне дослідження понад 200 мікрорайонів.

- Вивчення чинників впливу: Аналіз кореляції між ціною, кількістю відгуків та тривалістю оренди.

- Профілювання орендодавців: Класифікація власників житла на сегменти «приватних» і «професійних» гравців ринку.
 
- Візуалізація даних: Побудова графіків, таких як Boxplots, Violinplots і Heatmaps, для демонстрації розподілів та взаємозв'язків.
 
Технологічні інструменти:

- Мова програмування: Python.
 
- Бібліотеки: Pandas, Matplotlib, Seaborn.
  
- BI-системи: Tableau.
  
Результат проєкту: Зібрано 17 аналітичних блоків та створено 2 дашборди ([1](https://public.tableau.com/app/profile/a.a8255/viz/AirbnbNYC2019public/Dashboard), [2](https://public.tableau.com/app/profile/a.a8255/viz/AirbnbNYC2019cleandatapublic/Dashboard)), які допомагають відповісти на ключові питання щодо найприбутковіших локацій, середньої вартості житла та рівня активності клієнтів. Цей аналіз може слугувати надійною базою для прийняття бізнес-рішень про інвестування в нерухомість або розробку ефективних стратегій оренди.
